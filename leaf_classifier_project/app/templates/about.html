About Our System

Advanced AI-powered plant species identification using cutting-edge technology

Project Overview

Our Edge-Aided Plant Species Identification system combines advanced computer vision, machine learning, and deep learning techniques to accurately identify plant species from leaf images.

Computer Vision

Multiple edge detection algorithms analyze leaf structure, vein patterns, and morphological characteristics.

Machine Learning

Hybrid classification system combining traditional ML algorithms with deep learning models.

Feature Analysis

Comprehensive feature extraction including geometric, texture, color, vein, and margin characteristics.

Validation

Intelligent pre-filtering ensures only valid leaf images are processed for species identification.

Methodology

Data Preprocessing
  • • Grayscale conversion
  • • Image resizing (224×224)
  • • Noise injection testing
  • • Denoising filters
Edge Detection
  • • Sobel operator
  • • Prewitt filter
  • • Canny edge detection
  • • Laplacian of Gaussian
Feature Extraction
  • • Geometric descriptors
  • • Texture analysis (LBP, GLCM)
  • • Vein pattern analysis
  • • Margin characteristics
Classification
  • • K-Nearest Neighbors
  • • Support Vector Machines
  • • Random Forest
  • • Deep Neural Networks
Evaluation
  • • PSNR & SSIM metrics
  • • Confusion matrices
  • • ROC curves
  • • Processing time analysis
Validation
  • • Leaf detection
  • • Rule-based filtering
  • • ML-based validation
  • • Confidence scoring
